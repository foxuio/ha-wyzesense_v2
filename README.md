# ha-wyzesense_v2
Custom WyzeSense integration for Home Assistant with optimized detection.


## **Updates**

After rewriting the code, Home Assistant now supports Wyzesense with current 2025 updates.

I will post it very soon. I've called it Wyzense V2, this integration was initially designed by kevinvincent, credits to him.

I will continue to support it as long as possible.

### **Compatible devices:**

Wyzesense HUB v1

Door sensor V1 and V2 (Immediate status changes)

Motion sensors V1 and v2 (maintains the 40Sec. status update)
